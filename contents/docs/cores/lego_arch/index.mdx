# lego_arch
lego核心接口组件，与架构相关

## 简介

涉及与架构相关的代码，可以被分为三种功能类型的操作，其一，需要按位操作寄存器；其二，在代码中嵌入汇编指令；其三，处理trap逻辑。

基于以上三点，该核心组件的设计如下：

- 提供了便于按位操作寄存器的方法
- 将一些特殊指令包装为Rust函数
- 定义一系列的与架构trap相关的逻辑和接口
