# 概述

**欢迎来到操作系统开发——lego-os 探索使用Rust语言编写操作系统，采用组件化的开发方式**

开发操作系统是一项令人振奋但充满挑战的工作，作为一名对探索操作系统原理充满热情的开发者，
你或许渴望深入理解操作系统的本质以及它的强大功能，那么阅读内核源代码无疑是最直接且高效的学习途径。
在众多知名操作系统中，如 Linux、Windows、Android 和 macOS，Linux 凭借其强大的社区支持和完全
开源的特性，始终是开发者学习操作系统的理想平台。然而，仅仅学习理论或阅读他人代码始终难以真正领悟
操作系统的精髓。若想更深刻地理解操作系统，不妨亲自动手，从零开始编写一个简单的操作系统内核。

lego-os 是一个实验性项目，旨在探索使用现代高级编程语言 Rust 构建操作系统内核。我们采用组件化
的开发方式，将操作系统功能拆解为多个独立模块。在这一过程中，你不仅可以学习如何开发一个简单的系统
内核，还能通过实践加深对操作系统的理解。同时，我们也欢迎你的创意和建议，共同完善 lego-os。

接下来，我们将向你阐述为何选择 Rust 语言、为何采用组件化开发方式，以及我们的组件化方法是如何实现的。
在此过程中，我们还将借鉴和引用 Linux 内核中的一些优秀设计理念。准备好踏上这段探索之旅了吗？
