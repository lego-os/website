# 资料推荐

## Rust语言学习资料

- [The Rust Programming Language - The Rust Programming Language](https://doc.rust-lang.org/book/)：Rust 官方语言学习手册，带你熟悉Rust语言的语法和特性
- https://github.com/rust-lang/rustlings.git ：这个项目包含一些小练习，让你习惯阅读和编写Rust代码
- [Introduction - The Cargo Book](https://doc.rust-lang.org/cargo/index.html)：Rust包管理Cargo使用手册
- [What is rustc? - The rustc book](https://doc.rust-lang.org/rustc/index.html)：rustc编译器手册
- [core - Rust](https://doc.rust-lang.org/core/index.html) & [std - Rust](https://doc.rust-lang.org/std/index.html)：Rust语言核心库和标准库
- [Introduction - The Rust Reference](https://doc.rust-lang.org/reference/index.html)：描述每种语言结构及其用法、内存模型、并发模型、运行时服务、链接模型和调试工具、影响设计的语言的基本原理和参考
- [Introduction - The Rustonomicon](https://doc.rust-lang.org/nomicon/index.html#the-rustonomicon)：Unsafe Rust
- [The Unstable Book - The Rust Unstable Book](https://doc.rust-lang.org/nightly/unstable-book/index.html)：Nightly版本（非稳定版本）
- [Introduction - The Embedded Rust Book](https://doc.rust-lang.org/stable/embedded-book/)：Rust嵌入式开发

## RISC-V 手册

RISC-V github组织：[RISC-V](https://github.com/riscv)，几乎囊括了RISC-V的所有手册文档，比较常用的几个：

- [riscv/riscv-isa-manual: RISC-V Instruction Set Manual](https://github.com/riscv/riscv-isa-manual)：RISC-V非特权和特权指令集手册
- [riscv/riscv-plic-spec: PLIC Specification](https://github.com/riscv/riscv-plic-spec)：RISC-V平台级中断控制器
- [riscv/riscv-fast-interrupt: Proposal for a RISC-V Core-Local Interrupt Controller (CLIC)](https://github.com/riscv/riscv-fast-interrupt)：核心-本地中断控制器
- [riscv-non-isa/riscv-elf-psabi-doc: A RISC-V ELF psABI Document](https://github.com/riscv-non-isa/riscv-elf-psabi-doc)：ABI规范

## 常用工具

- [Documentation for binutils 2.43](https://sourceware.org/binutils/docs/)：Gnu binutils
- [GNU Make Manual - GNU Project - Free Software Foundation](https://www.gnu.org/software/make/manual/)：make工具
- [riscv-collab/riscv-gnu-toolchain: GNU toolchain for RISC-V, including GCC](https://github.com/riscv-collab/riscv-gnu-toolchain)：riscv toolchian
- [Linux man pages online](https://www.man7.org/linux/man-pages/)：Linux man手册
- [GDB Documentation](https://sourceware.org/gdb/documentation/)：gdb调试工具
- [QEMU](https://www.qemu.org/)：qemu模拟器

## 操作系统和IDE

推荐使用Ubuntu或Debian操作系统，也可以选择自己喜欢的发行版

- [Enterprise Open Source and Linux | Ubuntu](https://ubuntu.com/) 
- [Debian -- 通用操作系统](https://www.debian.org/)

IDE推荐使用vscode，当然也可以使用RustRover：

- [Visual Studio Code - Code Editing. Redefined](https://code.visualstudio.com/)
- [RustRover: Rust IDE by JetBrains](https://www.jetbrains.com/rust/)

## OS开发学习教程

- [Expanded Main Page - OSDev Wiki](https://wiki.osdev.org/Main_Page)：关于OS的一些知识
- [mit-pdos/xv6-riscv: Xv6 for RISC-V](https://github.com/mit-pdos/xv6-riscv)：xv6用于教学的操作系统，使用C语言编写，面向RISC-V架构
- [Writing an OS in Rust](https://os.phil-opp.com/)：本博客系列使用Rust编程语言创建了一个小型操作系统
- 《[Operating Systems: Three Easy Pieces](https://pages.cs.wisc.edu/~remzi/OSTEP/)》
